Indicator Add-In

Basic Information:

  • Always start with: '=Indicator.'
  • Enqoute Parmeter! E.g all indicators for Germany: GET_INDICATOR("Germany")

Following API calls are available:

  • GET_INDICATOR(indicator): Retrieves time series data for a specific indicator
  • GET_COUNTRY(country): Retrieves all indicators for a specific country
  • GET_INDICATOR_FOR_COUNTRY(indicator, country): Retrieves time series data for a specific indicator and country
  • GET_RECENT_INDICATOR(indicator): Retrieves the most recent values for a specific indicator
  • GET_RECENT_COUNTRY(country): Retrieves the most recent indicators for a specific country
  • GET_RECENT_INDICATOR_FOR_COUNTRY(indicator, country): Retrieves the most recent value for a specific indicator and country.

Problems:

  • Currently numbers might not be converted correctly. I.e inflation rate returns 20% instead of .2
  • Spelling is important! For countries: first letter needs to be capital: Germany. For indicators: usually lower case and - instead of empty spaces. The indicators can be found on https://tradingeconomics.com/indicators.
